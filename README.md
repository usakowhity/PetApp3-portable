# 🐾 PetApp3 Portable  
**Standalone Edition with Embedded Python**

---

## 🇬🇧 English Description

**PetApp3 Portable** is the standalone, fully self‑contained version of PetApp3.  
It includes an embedded Python environment and requires **no installation**.  
Simply extract the ZIP and run `run.bat`.

This edition is designed for users who want to use PetApp3 without installing Python or development tools.

---

### 📘 Features

- 15 pet states (n1–n3, p1–p12)  
- English prompt generation (first‑person perspective)  
- Species‑specific Joy behavior (dog / cat / rabbit)  
- Affection (p12) reacts to your custom affection word  
- Works entirely from the extracted folder  
- No installation required (ZIP only)  
- WhisperTinyEngine for voice recognition  
- Face detection (smile reaction)  

---

### 📘 How to Use

1. Download the ZIP from Releases.  
2. Extract it to any folder (PC, USB drive, SD card).  
3. Run **`run.bat`**.  
4. Register your pet’s profile (species, appearance, affection word).  
5. Assign images or videos to each state.  
6. Open the PlayWindow to see your pet react.

---

### 📘 System Requirements

- Windows 10 / Windows 11  
- No GPU required  
- Python included (portable)  
- Camera recommended (for smile detection)  
- Microphone recommended (for voice commands)

# 🇯🇵 日本語説明  
**PetApp3 Portable（ポータブル版）** は、Python をインストールせずに使える  
スタンドアロン版の PetApp3 です。  
Python 実行環境が同梱されており、ZIP を展開して `run.bat` を実行するだけで動作します。

開発環境を用意せずに PetApp3 を使いたいユーザー向けのエディションです。

---

### 📘 特徴

- 15種類のペット状態（n1〜n3、p1〜p12）  
- 英語プロンプト生成（1人称視点）  
- 種別ごとの「喜び」動作（犬／猫／ウサギ）  
- p12（愛情反応）はユーザー設定の「魔法のことば」に反応  
- ZIP展開のみで動作（インストール不要）  
- WhisperTinyEngine による音声認識  
- 顔検出による笑顔反応  

---

### 📘 使い方

1. ZIPファイルをダウンロードします。  
2. 任意のフォルダ（PC、USBメモリ、SDカードなど）に展開します。  
3. **`run.bat`** を実行します。  
4. ペットプロフィール（種別・外見・魔法のことば）を登録します。  
5. 各状態に画像または動画を割り当てます。  
6. PlayWindow を開くと、ペットがリアルタイムで反応します。

---

### 📘 動作環境

- Windows 10 / Windows 11  
- GPU不要  
- Python同梱（ポータブル動作）  
- カメラ推奨（笑顔検出）  
- マイク推奨（音声コマンド）
